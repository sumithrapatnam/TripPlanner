# 🌍 Trip Planner
### Plan Less. Travel More. ✈️

> "Every journey begins with a single plan."

Trip Planner is a smart web application that helps users organize their trips with ease. Instead of spending hours searching for destinations, creating schedules, and estimating costs, users can plan everything in one place.

## ✨ Features
- 🗺️ Personalized Trip Planning
- 📍 Explore Popular Destinations
- 📝 Day-wise Trip Itinerary
- 💰 Budget Planning
- 👨‍👩‍👧‍👦 Group Trip Support
- 📅 Easy Travel Scheduling
- 🎯 Simple & User-Friendly Interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## 🎯 Project Objective
Planning a trip often requires switching between multiple websites for destinations, budgeting, transportation, and scheduling. This project simplifies the entire process by providing everything in one platform, making travel planning faster, easier, and more enjoyable.
